# Trending Repos API and SPA

Used tech stack: Vite, React, Express, Mysql (with mysql2), Sequelize.

# the project is in progress...

how to run: 

open proj folder,

open terminal,

cd backend, 

npm install, 

node server.js, 

cd .., 

cd frontend, 

npm install, 

npm run dev, 

open localhost:5173 in browser,
